:::info
lib-sourcify is used to directly verify contracts, it's NOT a wrapper around the Sourcify API. It supports Node.js and browser environments.

Applications can integrate lib-sourcify in their frontends. This allows verifying contract sources on the user's end and minimizes trust towards Sourcify. You can find an example [here](/blog/recap-2025/#lib-sourcify-on-browser-local-verification).
:::
